# КОД - Кабінет Освітніх Даних

Інформаційна система університету для електронного документообігу та управління навчальними процесами.

## Про проєкт

Платформа електронного документообігу та навчального середовища ВНЗ з перспективою розвитку до LMS.

**Напрям:** Цифровізація освітніх процесів
